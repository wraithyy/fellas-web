# GF Začátečníci
Tato lekce je vhodná pro někoho, kdo se cvičením úplně začíná nebo se vrací po dlouhé pauze zpět ke cvičení. Lekce má menší kapacitu, a tak se každému můžeme důkladně věnovat.
